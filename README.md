
## express-perhaps

My very opinionated Express starting point.  

Installing
```
npm i
```

Running App
```
npm run start
```

Serving the pp
```
npm run serve
```

Testing
```
npm run test
```

:)
```
git clone git@github.com:DarrylD/express-perhaps.git && rm -rf .git
```

Notable Includes
- [x] Backpack (simple building)
- [x] Flow type (type checking)
- [x] Rxjs (async stuff)
- [x] Express (routing)
